# 💌 사이사이 v1.7.4 업데이트 안내

사용자분들의 더 나은 앱 사용을 위해
신규 기능 추가 및 사용성·안정성을 개선했습니다.

## ✨ 신규 기능

### 앨범 생성 개선
- 이미 등록된 날짜가 있는 경우 자동으로 **‘앨범 생성 / 미디어 추가’ 모드로 전환**되어 더 편리해졌어요. 💫

### 앨범 설명 더보기
- 이제 긴 설명도 **‘더보기’ 버튼**으로 전체 내용을 확인할 수 있어요. 🌷

## 🔧 버그 수정

### 위젯 D-DAY 계산 오류 수정
- 특별한 날의 **날짜 계산 정확도**를 개선했어요. 🎉

### 날짜 선택 오류 수정
- 일부 환경에서 **날짜가 올바르게 표시되지 않던 문제**를 해결했습니다.
