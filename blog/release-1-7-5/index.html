사이사이 v1.7.5 업데이트 안내

November 9, 2025 · One min read
사이사이 관리자
고객센터 운영팀

사이사이를 이용해주시는 모든 분들께 감사드립니다. 이번 업데이트에서는 새로운 기능이 추가되고, 기존 기능의 안정성이 더욱 향상되었습니다.

✨ 신규 기능​

달력 시작 요일 설정​

  • 달력의 시작 요일을 원하는 요일로 변경할 수 있어요. (일/월 등)

앱 내 비밀번호 변경​

  • 앱 설정에서 비밀번호를 직접 변경할 수 있도록 기능을 추가했어요.

사진 업로드 미리보기​

  • 사진 업로드 과정에서 미리보기로 선택한 이미지를 확인할 수 있어 실수가 줄어들어요.

앨범 사진 다운로드​

  • 앨범 속 사진을 기기에 저장(다운로드) 할 수 있는 기능을 추가했어요.

🐞 버그 수정​

동영상 썸네일 표시 문제 해결​

  • 동영상 업로드 시 썸네일이 표시되지 않던 문제를 수정했습니다.

📲 업데이트 방법​

  • iOS: App Store에서 업데이트
  • Android: Google Play에서 업데이트

💬 기타 안내​

사이사이는 앞으로도 여러분의 추억이 더 따뜻하고 안전하게 남겨질 수 있도록 꾸준히 개선해 나가겠습니다. 오늘도 사이사이와 함께 따뜻한 하루 보내세요 🌷

— 사이사이 팀 드림
